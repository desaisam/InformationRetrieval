The code for crawler is written in Python3.

===============================================================================================================
Install Beautiful Soup, Requests using follwing commands:  

> pip install bs4

> pip install requests

================================================================================================================

The folder contains following  files :

> Crawler_bfs.py --> Web crawler using breadth first search
> Crawler_dfs.py --> Web crawler using breadth first search
> Crawler_focused.py --> Focused web crawler using breadth firstt search
> keys.txt   --> Contains keys which is used for focused crawling
> duplicate_counter.txt --> Contains the count of duplicate urls, depth and urls generated from focused crawling
> bfs_crawler.txt --> Contains the crawled urls using breadth first search algorithm
> dfs_crawler.txt --> Contains the crawled urls using dephth first search algorithm
> task_1h.txt --> Explanation for Task 1 H
> task_2e.txt --> Explanatin for Task 2 E

================================================================================================================

How to compile ?
Install python in the machine, set the directories, navigate to this folder then: 
> python Crawler_bfs.py 
> python  Crawler_dfs.py 
> python  Crawler_focused.py

================================================================================================================ 

Maximum depth reached ?
> Depth First Search --> 6
> Bread First Search --> 2
> Focused Crawler --> 6
